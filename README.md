## English

## 📌 Overview

This project is a Python client for interacting with the **RIPE Database REST API**.

The purpose of this code is to simplify communication with the RIPE Database for **LIR (Local Internet Registry)** organizations.  
Using this client, LIRs can automate database operations such as updating and managing their RIPE objects through API requests.

Instead of manually editing objects from the RIPE Database web interface, this project allows network operators to perform changes programmatically using Python.

---

## 🚀 Features

- Connect to RIPE Database REST API
- Authenticate using RIPE API Keys
- Read RIPE Database objects
- Update RIPE objects programmatically
- Automate LIR database management
- Easy integration into network automation workflows
